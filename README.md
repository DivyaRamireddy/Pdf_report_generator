# PDF Report Generator

A simple Python project that generates a formatted PDF report from given data using the `reportlab` library.

---

## Features
- Creates professional-looking PDF reports
- Adds title, date, and structured content
- Uses standard A4 page size
- Easy to customize for invoices, certificates, or reports

---

## Concepts Used
- File generation
- reportlab library
- Page formatting
- Text positioning
- Dictionary-based data handling

---
